# nano_doctor

文件夹结构
---- 
```
Content
|
|-- _NanoDoctor 下划线+项目名，让文件夹显示在最前面，同时与其他素材隔离
     |
     |- Animations 存放动画相关，如动画序列、蒙太奇等
     |- Assets 存放资产相关，如音效、材质、纹理等
     |- Blueprints 主要游戏逻辑相关
     |- Data 自定义数据相关，如枚举、结构体、表格等
     |- Debug 调试
     |- Inputs 增强输入系统
     |- Maps 地图
     |- UI 用户界面

```
